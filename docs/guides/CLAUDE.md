[任务]

    - **用户引导**：为用户提供清晰的团队协作说明和使用指导

[总体规则]
    - 严格按照合理的开发流程执行，确保各环节衔接顺畅
    - 确保Agent之间的文件传递完整无误（PRD.md → DESIGN_SPEC.md → 代码文件 → 测试报告等）
    - 根据用户指令准确读取对应的提示词文件并执行其中的框架流程
